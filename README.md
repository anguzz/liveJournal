# Live Journal

This project began in **2023** as a **SvelteKit + Skeleton Labs** site deployed on **Vercel**.

Due to dependency churn (Skeleton v3 → v4, Tailwind changes, Node version issues) and the fact that the site only contained static content, the project was simplified to **plain HTML, CSS, and JavaScript** with no build step.

## Current Status

As of **2026**, the Live Journal has been merged into my main site:

https://angs.dev/music

This repository is kept for historical reference.
